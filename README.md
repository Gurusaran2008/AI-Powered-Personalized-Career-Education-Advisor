# AI Career Advisor

AI Career Advisor is a smart web application developed using Flask that helps students and job seekers discover suitable career paths, analyze resumes, and prepare for interviews. The project combines AI-inspired logic with a user-friendly interface to provide personalized career guidance.

## Project Overview
This project is designed to assist users in:
- identifying career options based on skills and interests
- receiving career recommendations and roadmaps
- uploading resumes for analysis and feedback
- accessing interview preparation resources and course suggestions

## Key Features
- User registration and login system
- Career prediction and roadmap guidance
- Resume upload and analysis support
- ATS-style resume feedback suggestions
- Interview preparation resources
- Responsive and modern UI

## Technologies Used
- Python
- Flask
- SQLite
- HTML/CSS/JavaScript
- pdfplumber
- python-docx
- Werkzeug

## Project Structure
- app.py - Main Flask application
- templates/ - HTML pages for the website
- static/ - CSS and static assets
- uploads/ - Uploaded resume files
- users.db - SQLite database for user accounts

## Installation
1. Install Python 3.10 or above
2. Install dependencies:
   pip install -r requirements.txt
3. Run the application:
   python app.py
4. Open the browser at:
   http://127.0.0.1:5000

## How It Works
1. The user registers or logs in
2. The user fills the career assessment form
3. The system generates career recommendations
4. The user can upload a resume for analysis and improvement tips
5. Interview and course resources are provided for further guidance
